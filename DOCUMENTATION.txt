üöÄ HOW TO USE
=============

1. CUSTOMER JOURNEY:
   a) Visit services.html
   b) Browse available services
   c) Click "Enquire Now" on desired service

   c) Update inquiry statuses
   d) Contact customers directly
   e) Track project progress

3. INQUIRY MANAGEMENT:
   a) Go to inquiry-dashboard.html
   b) View all inquiries with real-time stats
   c) Search and filter inquiries
   d) Click "View Details" to see full information
   e) Update inquiry status
   f) Send replies via email/SMS
   g) Export inquiries as CSV

üîß CUSTOMIZATION GUIDE
=======================

To Add New Service:
1. Go to admin-services.html
2. Click "Add New Service" button
3. Fill form with:
   - Service name
   - Category (Website Design / Special Features / Support)
   - Icon (Font Awesome class or URL)
   - Description
   - Features (comma-separated)
   - Price
   - Delivery timeline
   - Status (Published/Draft)
4. Save service
5. Service appears on services.html immediately

To Customize Colors:
1. Open styles.css
2. Modify CSS variables in :root section
3. Update these colors:
   - --primary-color: #10b981
   - --secondary-color: #ffffff
   - --accent-color: #3b82f6

To Add New Service Category:
1. Go to admin-services.html ‚Üí Settings
2. Enter category name
3. Click "Add Category"
4. Category available for new services

==================

üÜò TROUBLESHOOTING
==================

Issue: Services not appearing
Solution: Check browser console for errors, ensure JavaScript is enabled

Issue: Inquiries not saving
Solution: Check localStorage is enabled, clear browser cache and reload

Issue: Modal not closing
Solution: Try clicking outside modal or close button, refresh page if stuck

Issue: Styling looks broken
Solution: Hard refresh (Ctrl+Shift+R), clear CSS cache
