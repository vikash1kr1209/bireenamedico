Quick Start Guide

For Customers: Visit services.html to browse pharmacy website services and submit inquiries.

For Admin: Go to admin-services.html to add services and manage the platform.

For Inquiry Management: Use inquiry-dashboard.html to track and respond to customer inquiries.

# Quick Links: - Main Website: http://localhost:8000 - Services: http://localhost:8000/services.html - Admin Panel: http://localhost:8000/admin-services.html - Inquiries: http://localhost:8000/inquiry-dashboard.html

Key Features

Service Management: Add, edit, and manage pharmacy website development services with pricing, timelines, and categories.

Customer Inquiries: Comprehensive inquiry form with feature selection, budget range, and file uploads.

Status Tracking: Track inquiries through workflow: New → Contacted → Proposal Sent → In Progress → Completed.

Communication: Reply to customers, send updates, and manage all communications in one place.

Analytics: Real-time statistics, search, filtering, and CSV export capabilities.

Responsive Design: Fully optimized for mobile, tablet, and desktop devices.
